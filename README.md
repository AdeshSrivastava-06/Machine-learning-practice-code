A structured collection of Python scripts covering Machine Learning, Feature Engineering, and Data Science concepts.

---

## Structure

| Folder | Description |
|---|---|
| 01-feature-engineering | Encoding, scaling, feature selection, transformation |
| 02-eda-and-stats | Exploratory data analysis, visualization, hypothesis testing |
| 03-supervised-learning | Classification and regression with scikit-learn |
| 04-unsupervised-learning | Clustering, dimensionality reduction |
| 05-data-preprocessing | Null handling, outlier treatment, data cleaning |

---

## Tech Stack

- Python 3.x
- NumPy, Pandas, Scikit-learn
- Matplotlib, Seaborn
