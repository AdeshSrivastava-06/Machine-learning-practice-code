main topic curse of dimensionality
->> Curse of Dimensionality
Main Concept

The curse of dimensionality refers to the problems that arise when the number of features (columns / dimensions) in a dataset becomes very large.

As we add more and more features than required, the data space increases exponentially. Because of this, the data points become sparse, distances between points lose meaning, and model accuracy decreases instead of increasing.

In short:
More features ≠ better model
 disadvantages-> 1.Performance Decreases
                 2.Computation
 how to solve-> Dimensionality reduction the  2 methods are feature Selection(Forward selection and Backward elimination) and feature Extraction(PCA,LDA,TSNE)

the covariance matrix of X1,X2 will be
X1[var(X1)     cov(X2,X1)]

       X1         X2
cov(X1,X1) = var(X1)
cov(X1,X2) = cov(X2,X1) so the matrix are symmetric
