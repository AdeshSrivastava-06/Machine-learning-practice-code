# Unsupervised Learning

Unsupervised learning is a type of machine learning where the model is trained on data that has no labels. The algorithm tries to find hidden patterns or structure in the data on its own, without any guidance on what the correct output should be.

It is mainly used for two purposes: clustering, where the goal is to group similar data points together, and dimensionality reduction, where the goal is to compress data into fewer features while retaining the most important information.

Common algorithms include K-Means and DBSCAN for clustering, and PCA (Principal Component Analysis) for dimensionality reduction.

It is widely used in customer segmentation, anomaly detection, recommendation systems, and data compression.
